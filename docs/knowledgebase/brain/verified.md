## Key Claims Verified (via WebSearch 2026-01-10)

### 1. Thalamic Driver/Modulator Distinction
**Status:** VERIFIED (2025 literature confirms)

WebSearch confirmed:
- Driver inputs (Class II axons, large RL terminals): 5-10% of synapses, determine receptive field properties
- Modulator inputs (Class I axons, small RS terminals): 90-95% of synapses, alter gain of signal transmission
- Retinal input to LGN is driver; cortical L6 feedback is modulator
- **2025 Update (Salk Institute, J. Neuroscience):** L5CT drivers are feedforward; L6CT modulators follow reciprocal bias
- **2021-2025 Research:** Layer 5 CT projections drive visual responses in pulvinar (not just layer 6)

Sources:
- [Complementary Organization of Mouse Driver and Modulator Cortico-thalamo-cortical Circuits (2025)](https://pubmed.ncbi.nlm.nih.gov/39824633/)
- [Thalamic Circuit Diversity: Modulation of the Driver/Modulator Framework](https://pubmed.ncbi.nlm.nih.gov/26793068/)
- [Distinct driving vs modulatory influences of visual corticothalamic pathways](https://pubmed.ncbi.nlm.nih.gov/34614389/)

### 2. Cortex Layer Feedforward/Feedback Rules
**Status:** VERIFIED (2025 literature confirms)

WebSearch confirmed:
- Feedforward projections: originate from supragranular layers, target layer 4
- Feedback projections: stem from infragranular layers, target supra- and infragranular layers while avoiding layer 4
- **2025 Research:** Layer 6 corticocortical neurons identified as major route for intra- and interhemispheric feedback
- **Frequency signatures:** Feedforward = gamma (30-70 Hz); Feedback = alpha/low-beta (8-15 Hz)

Sources:
- [Layer 6 corticocortical neurons are a major route for feedback (2025)](https://elifesciences.org/articles/100478/peer-reviews)
- [Layer-specific spatiotemporal dynamics of feedforward and feedback (2025)](https://pubmed.ncbi.nlm.nih.gov/40462954/)
- [Modelling Predictive Coding in V1: Layer 2/3 Circuits (2025)](https://www.biorxiv.org/content/10.1101/2025.11.01.686040v1.full)

### 3. Basal Ganglia Direct/Indirect/Hyperdirect Pathways
**Status:** VERIFIED (2025 literature confirms with updates)

WebSearch confirmed:
- **Direct (GO):** Striatum D1 -> GPi/SNr (GABA) -> Thalamus = disinhibition
- **Indirect (NO-GO):** Striatum D2 -> GPe (GABA) -> STN -> GPi -> Thalamus = increased inhibition
- **Hyperdirect (STOP):** Cortex -> STN directly (Glu) -> GPi/SNr = fast, global stop

**2025 Updates from Graybiel (Movement Disorders):**
- Fresh evidence of "formerly unappreciated pair of direct and indirect pathways"
- The Witten group demonstrated pathway-specific functions can be changed/reversed by learning
- Center-surround-context "Triple-control" model proposed: direct (center), indirect (surround + context)

Sources:
- [Surprises From the Basal Ganglia: Stop and Go Have New Meaning (Graybiel 2025)](https://movementdisorders.onlinelibrary.wiley.com/doi/full/10.1002/mds.70008)
- [Direct and indirect pathways: a critical reappraisal](https://pubmed.ncbi.nlm.nih.gov/25065439/)
- [Multiple dynamic interactions from BG direct and indirect pathways](https://pmc.ncbi.nlm.nih.gov/articles/PMC10522336/)

### 4. TRN Gating Mechanism
**Status:** VERIFIED (2025 literature confirms)

WebSearch confirmed:
- TRN is "guardian of the gateway" - shapes which thalamic signals reach cortex
- Receives collaterals from both thalamocortical and corticothalamic axons
- Sends GABAergic inhibition back to relay nuclei
- **2025 Update:** Dual inhibitory network identified - GPe inhibition to all TRN neurons, plus intra-TRN inhibition (SOM->PV)
- **2025 Update:** Two transcriptionally defined TRN populations (Ecel1+ and Spp1+) with different burst firing properties

Sources:
- [Dual Inhibitory Network in TRN (2025)](https://www.biorxiv.org/content/10.1101/2025.10.05.680571v1.full)
- [Biophysical modeling of TRN subpopulations (2025)](https://pmc.ncbi.nlm.nih.gov/articles/PMC12454877/)
- [TRN: structure, function and concept](https://pubmed.ncbi.nlm.nih.gov/15297152/)

### 5. Cerebellar Circuitry
**Status:** VERIFIED (2025 literature confirms with updates)

WebSearch confirmed:
- Three-layer cortex (molecular, Purkinje, granule) - correct
- Mossy fibers from multiple sources -> granule cells -> parallel fibers -> Purkinje cells
- Climbing fibers exclusively from inferior olive, 1:1 with Purkinje cells, error signal
- Purkinje cells ONLY output of cerebellar cortex, GABAergic, inhibit deep nuclei

**2025 Updates (Nature Reviews Neuroscience):**
- Cerebellum learns basic associative feedforward control policies
- Population coding, newly discovered circuitry, diverse plasticity rules suggest multiple parallel modules
- Cerebellar-cortical beta oscillations emerge 2 seconds before action as predictive signal
- Cerebellum initially drives prefrontal activity during pre-movement, then reverses near movement onset

Sources:
- [Cerebellar circuit computations for predictive motor control (Nature Reviews Neuroscience 2025)](https://www.nature.com/articles/s41583-025-00936-z)
- [The cerebellar connectome (2025)](https://www.sciencedirect.com/science/article/pii/S0166432825000439)
- [Cerebellar-cortical beta oscillations (2025)](https://www.sciencedirect.com/science/article/abs/pii/S0960982225015593)

### 6. Four Major Basal Ganglia Loops
**Status:** VERIFIED

The basal ganglia documentation correctly describes:
1. **Motor Loop:** M1, premotor, SMA -> Putamen -> GPi -> VL -> Motor cortex
2. **Oculomotor Loop:** FEF, SEF -> Caudate (body) -> SNr -> VA/MD -> FEF
3. **Cognitive Loop:** DLPFC, PPC -> Caudate (head) -> GPi -> VA/MD -> DLPFC
4. **Limbic Loop:** OFC, ACC, vmPFC -> NAcc -> Ventral pallidum -> MD -> OFC/ACC

### 7. Papez Circuit
**Status:** VERIFIED

Hippocampus -> Fornix -> Mammillary bodies -> Mammillothalamic tract -> Anterior thalamus -> Cingulate cortex -> Cingulum -> Parahippocampal gyrus -> Entorhinal cortex -> Hippocampus

### Sherman & Usrey 2024 Key Findings (Direct from PDF)

This paper is highly relevant to the brain-mimc architecture. Key confirmations:

1. **Driver vs Modulator (Class 1 vs Class 2):** Confirmed. Class 1 (drivers) have large boutons, high probability of release, ionotropic only. Class 2 (modulators) have small boutons, low release, metabotropic receptors.

2. **L5 CT = Drivers, L6 CT = Modulators:** Confirmed. L5 corticothalamic projections provide driver input to higher-order thalamus. L6 CT provides modulatory feedback.

3. **Transthalamic Routes:** Confirmed. Higher-order thalamic nuclei (pulvinar, MD, LP) enable cortex-to-cortex communication via L5 → HO thalamus → target cortex.

4. **First-order vs Higher-order Nuclei:** Confirmed. First-order receives peripheral drivers. Higher-order receives cortical L5 drivers.
