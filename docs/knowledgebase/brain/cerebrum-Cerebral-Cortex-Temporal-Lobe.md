# Region-Centric Communication Map: Cerebrum — Cerebral Cortex — Temporal Lobe

This document provides a complete communication map for every major region of the temporal lobe. For each region, it specifies what that area communicates with at every level: within the temporal lobe, with other cortical lobes, with subcortical structures, and with brainstem structures — all with layer-level connectivity where applicable.

---

## Foundational Reference

**Before reading this document, ensure familiarity with the Canonical Cortical Microcircuit document** (`cerebrum-Cerebral-Cortex-Canonical-Microcircuit.md`), which describes:
- The six cortical layers and their cell types
- Feedforward connections: Layer III → Layer IV of target
- Feedback connections: Layer V/VI → Layer I of target (avoiding Layer IV)
- Layer IV as the input layer; Layer V as subcortical output; Layer VI as thalamic feedback
- Granular vs agranular cortex types

The temporal lobe contains **granular cortex** (primary auditory cortex with thick Layer IV) and **dysgranular cortex** (association areas). The medial temporal lobe structures (hippocampus, entorhinal cortex) have distinct architectures covered in the Limbic Structures document.

## Part 1: Overview of the Temporal Lobe

### Anatomical Location

The temporal lobe occupies the lateral-inferior portion of the cerebral hemisphere. Key boundaries:

**Superior:** Lateral (Sylvian) fissure separates temporal from frontal and parietal lobes
**Posterior:** No sharp boundary; blends with occipital and parietal lobes
**Medial:** Extends to medial surface; includes hippocampal formation
**Inferior:** Rests on middle cranial fossa

### Major Gyri

**Superior Temporal Gyrus (STG):** Contains auditory cortex and Wernicke's area
**Middle Temporal Gyrus (MTG):** Semantic processing, motion (MT/V5)
**Inferior Temporal Gyrus (ITG):** Object recognition, ventral visual stream
**Fusiform Gyrus:** Face and word recognition (ventral surface)
**Parahippocampal Gyrus:** Scene recognition, gateway to hippocampus (medial surface)

### Functional Overview

The temporal lobe serves four major functions:

**Auditory Processing:** Primary reception and analysis of sound

**Language Comprehension:** Understanding spoken and written language (left hemisphere dominant)

**Visual Object Recognition:** "What" pathway — identifying objects, faces, scenes (ventral stream)

**Memory:** Medial temporal structures (hippocampus, entorhinal) critical for declarative memory

## Part 2: Primary Auditory Cortex (A1) — Core Auditory Region

**Location:** Heschl's gyrus (transverse temporal gyrus), hidden within the lateral sulcus on the superior surface of the temporal lobe

**Brodmann Area:** 41 (core)

**Function:** First cortical stage of auditory processing. Receives tonotopically organized input from thalamus. Extracts basic auditory features: frequency, intensity, timing.

### Tonotopic Organization

A1 contains a frequency map (tonotopy):
- **Posterior-medial:** Low frequencies
- **Anterior-lateral:** High frequencies
- Multiple tonotopic maps in core and belt regions

### Laminar Organization

A1 is **granular cortex** with thick Layer IV (heavy thalamic input, similar to V1 and S1).

**Layer IV:** Thick; receives MGN thalamic input; contains stellate cells
**Layer III:** Strong horizontal connections; outputs to belt areas
**Layer V:** Subcortical output

### Within Temporal Lobe

**To Belt Areas (Secondary Auditory):**
- **Layer III → Belt Layer IV**
- Surrounds A1
- Lateral belt, medial belt, parabelt

**To Parabelt:**
- Via belt areas
- Higher auditory processing

**To Superior Temporal Gyrus (Posterior):**
- Language-related areas

**From Belt Areas (Feedback):**
- Belt Layer V/VI → A1 Layer I

### With Thalamus

**From MGN (Medial Geniculate Nucleus):**
- MAJOR INPUT — driver input
- MGN (ventral division) → A1 Layer IV
- Tonotopically organized
- Lemniscal pathway (precise frequency tuning)

**To MGN (Feedback):**
- **Layer VI → MGN (ventral)**
- Modulates thalamic relay
- Attention effects

**From MGN (Dorsal and Medial Divisions):**
- To belt areas (less precise tuning)
- Multisensory, attention

**To/From Pulvinar:**
- Minor auditory connections

**To TRN:**
- Layer VI collaterals → TRN (auditory sector)
- Gating auditory input

### With Frontal Lobe

**To Prefrontal Cortex:**
- Via belt and parabelt
- Auditory working memory

**To Broca's Area (Left Hemisphere):**
- Via dorsal auditory stream
- Speech processing

### With Parietal Lobe

**To Posterior Parietal:**
- Auditory spatial information
- "Where" auditory stream

### With Subcortical Structures

**To Inferior Colliculus (Feedback):**
- **Layer V → Inferior colliculus**
- Corticocollicular feedback
- Modulates subcortical auditory processing

**To Striatum:**
- **Layer V → Striatum**
- Auditory input to basal ganglia

**To Pontine Nuclei:**
- **Layer V → Pons → Cerebellum**
- Auditory-motor coordination

**To Amygdala:**
- Emotional sounds
- Fear-related auditory stimuli

### From Brainstem (Neuromodulation)

**From Locus Coeruleus:**
- Noradrenergic modulation
- Attention, arousal

**From Raphe Nuclei:**
- Serotonergic modulation

**From Basal Forebrain:**
- Cholinergic (nucleus basalis)
- Plasticity, learning

### Functional Properties

**Frequency Tuning:**
- Neurons tuned to specific frequencies
- Narrow tuning (octave or less)

**Temporal Processing:**
- Rapid temporal modulation
- Important for speech perception

**Binaural Processing:**
- Integrates both ears
- Sound localization cues (ITD, ILD)

**Spectrotemporal Receptive Fields:**
- Respond to specific patterns over time and frequency

## Part 3: Secondary Auditory Cortex — Belt and Parabelt

**Location:** Surrounding A1; belt immediately adjacent, parabelt on lateral STG

**Brodmann Area:** 42 (parts), 22 (parts)

**Function:** Higher-order auditory processing. Analyzes complex sounds, auditory objects, patterns. Beginning of auditory "what" and "where" streams.

### Belt Areas

**Lateral Belt:** More lateral on STG
- Complex sound features
- Spectral patterns

**Medial Belt:** More medial
- Temporal patterns
- Some spatial processing

### Parabelt

**Location:** Lateral to belt, on exposed STG surface

**Function:** Highest level of auditory cortex hierarchy before association areas

### Within Temporal Lobe

**From A1:**
- A1 Layer III → Belt Layer IV
- Tonotopic but broader tuning

**From Belt to Parabelt:**
- Belt Layer III → Parabelt Layer IV

**To Superior Temporal Sulcus (STS):**
- Voice, biological sounds
- Multimodal

**To Anterior Temporal:**
- "What" stream (auditory object identity)

**To A1 (Feedback):**
- Layer V/VI → A1 Layer I

### Auditory Streams

**Anterior/Ventral Stream ("What"):**
- A1 → Belt → Parabelt → Anterior STG → Anterior temporal
- Auditory object identification
- "What is this sound?"

**Posterior/Dorsal Stream ("Where/How"):**
- A1 → Belt → Posterior STG/STS → Parietal → Frontal
- Spatial location of sounds
- Auditory-motor integration
- Speech articulation

### With Frontal Lobe

**To Prefrontal Cortex:**
- **Layer III → PFC Layer IV**
- Auditory working memory
- Executive control of auditory processing

**To Premotor Cortex:**
- Via dorsal stream
- Auditory-motor (speech, music)

**To Broca's Area:**
- Speech processing (dorsal stream)

### With Parietal Lobe

**To Posterior Parietal Cortex:**
- **Layer III → PPC**
- Auditory spatial
- Dorsal stream

**To Inferior Parietal (Supramarginal Gyrus):**
- Phonological processing
- Speech sounds

### With Thalamus

**From MGN (Dorsal, Medial Divisions):**
- Less precise tuning
- Multisensory
- Layer IV

**To MGN:**
- Layer VI feedback

**From Pulvinar:**
- Attention modulation

### Functional Properties

**Auditory Objects:**
- Recognizing sounds as distinct sources
- Voice, instruments, environmental sounds

**Pitch Processing:**
- Melody, harmonic structure
- Pitch center in lateral Heschl's gyrus

**Sound Categories:**
- Speech vs non-speech
- Music vs noise

## Part 4: Superior Temporal Sulcus (STS)

**Location:** Sulcus between superior and middle temporal gyri

**Function:** Multimodal integration, biological motion, social perception, voice processing. Critical for social cognition.

### Within Temporal Lobe

**From Auditory Areas:**
- Belt, parabelt → STS
- Voice, speech sounds

**From Visual Areas:**
- MT, MST → STS
- Biological motion, faces

**To Temporal Pole:**
- Social, emotional processing

**To Fusiform Gyrus:**
- Face processing

### With Parietal Lobe

**To/From Posterior Parietal (IPL, TPJ):**
- **Bidirectional**
- Theory of mind
- Attention

### With Frontal Lobe

**To/From Prefrontal Cortex:**
- Social cognition
- Executive control

**To/From Inferior Frontal Gyrus:**
- Mirror neuron system
- Action observation

### With Limbic Structures

**To/From Amygdala:**
- Social-emotional processing
- Face emotion

**To/From Hippocampus:**
- Social memory

### Functional Properties

**Biological Motion:**
- Point-light walkers
- Body movement

**Voice Processing:**
- Voice identity
- Vocal emotion

**Gaze Direction:**
- Where others are looking
- Social attention

**Audiovisual Integration:**
- Speech (lips + voice)
- McGurk effect

**Theory of Mind:**
- Understanding others' mental states
- Posterior STS/TPJ especially

## Part 5: Wernicke's Area — Posterior Superior Temporal Gyrus

**Location:** Posterior part of superior temporal gyrus (left hemisphere in most people); area surrounding posterior end of lateral sulcus

**Brodmann Area:** 22 (posterior part)

**Function:** Language comprehension. Understanding spoken language, word meanings, sentence structure. Part of the language network.

### Note on Modern Understanding

Classical Wernicke's area is now understood as part of a distributed language network. The posterior STG/STS region (BA 22) contributes to phonological processing (sound structure of language), while semantic processing (meanings) involves a broader network including MTG and parietal regions (angular gyrus BA 39, supramarginal gyrus BA 40). These parietal areas are not part of classical Wernicke's area but form the extended posterior language network.

### Within Temporal Lobe

**From Auditory Cortex:**
- Belt, parabelt → posterior STG/STS
- Phonological input

**To Middle Temporal Gyrus:**
- **Layer III → MTG Layer IV**
- Semantic processing

**To Anterior Temporal Lobe:**
- Sentence-level processing

**To STS:**
- Multisensory language

### With Parietal Lobe

**To/From Supramarginal Gyrus (Left):**
- **Bidirectional**
- Phonological processing
- Articulatory rehearsal

**To/From Angular Gyrus:**
- Semantic processing
- Reading

### With Frontal Lobe

**To/From Broca's Area:**
- **Arcuate fasciculus** (major white matter tract)
- Dorsal language stream
- Phonological → articulatory
- **Layer III ↔ Layer IV bidirectional**

**To/From DLPFC:**
- Language working memory
- Sentence processing

**To/From Inferior Frontal Gyrus:**
- Syntactic processing

### With Thalamus

**From Pulvinar:**
- Attention to language

**From MGN (medial division):**
- Some language-relevant auditory

### Functional Properties

**Phonological Processing:**
- Sound structure of words
- Phoneme discrimination

**Word Recognition:**
- Mapping sounds to words
- Lexical access

**Sentence Comprehension:**
- Syntactic processing
- Integration across words

**Deficits (Wernicke's Aphasia):**
- Fluent but meaningless speech
- Impaired comprehension
- Neologisms, paraphasias
- Often unaware of deficit

## Part 6: Middle Temporal Gyrus (MTG)

**Location:** Between superior and inferior temporal gyri

**Brodmann Area:** 21

**Function:** Semantic processing — word meanings, conceptual knowledge. Also contains visual motion area (MT/V5) at posterior end.

### Posterior MTG (Including MT/V5)

See Occipital Lobe document for detailed MT/V5 connectivity.

**Visual Motion Area MT:**
- Located at temporo-occipital junction
- Motion processing (dorsal stream)

### Anterior/Central MTG

**Function:** Semantic hub — word meanings, conceptual knowledge

### Within Temporal Lobe

**From Superior Temporal Gyrus:**
- Phonological → semantic
- STG Layer III → MTG Layer IV

**From Inferior Temporal:**
- Visual object information
- Object names

**To/From Temporal Pole:**
- Abstract semantics

**To/From Fusiform:**
- Category-specific knowledge

### With Parietal Lobe

**To/From Angular Gyrus:**
- **Bidirectional**
- Semantic integration
- Strong connection

**To/From Supramarginal Gyrus:**
- Phonological-semantic interface

### With Frontal Lobe

**To/From Inferior Frontal Gyrus:**
- **Layer III ↔ IFG**
- Semantic retrieval
- Selection among meanings

**To/From DLPFC:**
- Semantic working memory

**To/From Ventral PFC:**
- Semantic control

### With Limbic Structures

**To/From Hippocampus:**
- Via parahippocampal
- Semantic memory encoding

### Functional Properties

**Lexical Semantics:**
- Word meanings
- Synonyms, categories

**Conceptual Knowledge:**
- Facts about the world
- Associations

**Semantic Control:**
- Retrieving appropriate meaning
- Context-dependent interpretation

**Deficits (Lesions):**
- Semantic paraphasias
- Word-finding difficulties
- Category-specific deficits possible

## Part 7: Inferior Temporal Cortex (IT) — TEO and TE

**Location:** Inferior surface and lateral inferior aspect of temporal lobe; anterior to occipital lobe

**Brodmann Areas:** 20, 37 (parts)

**Function:** Highest level of ventral visual stream. Object recognition — identifies what objects are regardless of position, size, or viewpoint. Complex visual features, categories.

### Subdivisions

**TEO (Posterior IT):**
- More posterior
- Intermediate complexity features
- Retinotopic remnants

**TE (Anterior IT):**
- More anterior
- Complex objects, faces
- View-invariant representations

### Within Temporal Lobe

**To/From MTG:**
- Category semantics

**To/From Fusiform Gyrus:**
- Face, word processing

**To/From Temporal Pole:**
- Person identity
- Abstract concepts

**To/From Parahippocampal Cortex:**
- Scene processing
- Memory encoding gateway

### With Occipital Lobe

**From V4:**
- **V4 Layer III → IT Layer IV**
- Color, intermediate form
- Major feedforward input

**From LOC:**
- Shape information

**To V4 (Feedback):**
- **Layer V/VI → V4 Layer I**
- Object-based attention

**To V1, V2 (Feedback):**
- Via intermediate areas
- Long-range feedback

### With Frontal Lobe

**To Prefrontal Cortex:**
- **Layer III → PFC Layer IV**
- Object information for working memory
- Visual categorization

**To Orbitofrontal Cortex:**
- Object value
- Reward associations

**To Inferior Frontal:**
- Object naming
- Semantic retrieval

**From Prefrontal (Feedback):**
- Top-down object expectations
- Category search

### With Parietal Lobe

**To/From Posterior Parietal:**
- Some ventral-dorsal stream interaction
- Object-based attention

### With Limbic Structures

**To Amygdala:**
- **Layer III → Amygdala (basal nucleus)**
- Emotional significance of objects
- Faces especially

**To Perirhinal Cortex:**
- **Layer III → Perirhinal**
- Object memory
- Familiarity

**To Hippocampus (via Entorhinal):**
- Object information for episodic memory

### With Thalamus

**From Pulvinar:**
- **Pulvinar → IT Layer IV, I**
- Attention modulation
- Strong in IT

**To Pulvinar:**
- **Layer V → Pulvinar**
- Transthalamic

### With Subcortical Structures

**To Striatum:**
- **Layer V → Tail of caudate**
- Visual for habit learning

**To Claustrum:**
- Layer V → Claustrum

### Functional Properties

**Object Recognition:**
- Identifies objects across transformations
- Position invariance (partial)
- Size invariance
- View invariance (develops through hierarchy)

**Category Selectivity:**
- Face patches (patches selective for faces)
- Body patches
- Place-selective regions
- Word-selective (VWFA)

**Sparse Coding:**
- Individual neurons selective for specific objects/features
- Population coding for fine discrimination

**Receptive Fields:**
- Large (many degrees)
- Often bilateral (includes both visual fields)

## Part 8: Fusiform Gyrus — Face and Word Areas

**Location:** Ventral surface of temporal lobe, between parahippocampal gyrus (medial) and inferior temporal gyrus (lateral)

**Brodmann Area:** 37

**Function:** Contains specialized regions for face recognition (FFA) and visual word recognition (VWFA).

### Fusiform Face Area (FFA)

**Location:** Mid-lateral fusiform gyrus; lateral to the parahippocampal place area (PPA); right hemisphere dominant for faces

**Function:** Face recognition — identifies individual faces

#### Within Temporal Lobe

**From IT/TEO:**
- Complex visual features

**To/From STS:**
- Facial expression, gaze
- Dynamic faces

**To/From Temporal Pole:**
- Person identity
- Familiar faces

**To/From Amygdala:**
- Face emotion
- Threat detection

#### With Occipital Lobe

**From Occipital Face Area (OFA):**
- Earlier face processing
- Face parts

**From V4:**
- Color, form

#### With Frontal Lobe

**To Prefrontal Cortex:**
- Face working memory
- Person identification

**To OFC:**
- Face value, attractiveness

#### Functional Properties

**Face Selectivity:**
- Much stronger response to faces than objects
- Individual face discrimination

**Holistic Processing:**
- Processes faces as wholes
- Inversion effect (upside-down faces harder)

**Expertise Effects:**
- Can develop for other categories with expertise
- Car experts, bird experts

**Prosopagnosia:**
- Face blindness from FFA damage
- Cannot recognize faces (use voice, hair, etc.)

### Visual Word Form Area (VWFA)

**Location:** Left fusiform gyrus, posterior to FFA

**Brodmann Area:** 37 (left)

**Function:** Visual word recognition — recognizes written words

#### Within Temporal Lobe

**To/From Language Areas:**
- STG, MTG
- Sound and meaning

#### With Occipital Lobe

**From Visual Cortex:**
- Letter features
- Orthographic processing

#### With Parietal Lobe

**To/From Angular Gyrus:**
- Reading network
- Phonological-orthographic mapping

#### With Frontal Lobe

**To/From Broca's Area:**
- Reading aloud
- Phonological output

**To/From DLPFC:**
- Reading comprehension

#### Functional Properties

**Word Selectivity:**
- Responds to words > consonant strings > faces
- Left lateralized

**Learned Through Literacy:**
- Not present in illiterates
- Develops with reading acquisition

**Orthographic Processing:**
- Abstract letter identity
- Position coding

**Alexia:**
- Pure alexia from VWFA damage
- Cannot read but can write

## Part 9: Parahippocampal Gyrus — Scene and Spatial Processing

**Location:** Medial surface of temporal lobe, surrounding hippocampus

**Brodmann Areas:** 27, 28, 35, 36

**Contains:** Parahippocampal Place Area (PPA), Entorhinal Cortex, Perirhinal Cortex

### Parahippocampal Place Area (PPA)

**Location:** Posterior parahippocampal cortex

**Function:** Scene and place recognition — identifies spatial layouts, environments

#### Within Temporal Lobe

**From IT:**
- Object information

**To/From Retrosplenial Cortex:**
- Navigation, orientation

**To Hippocampus:**
- Via entorhinal cortex
- Spatial memory

#### With Occipital Lobe

**From Visual Cortex (LOC, V3, V4):**
- Scene structure
- Spatial layout

#### With Parietal Lobe

**To/From Posterior Parietal:**
- Spatial navigation
- Viewpoint

**To/From Precuneus:**
- Scene imagery
- Spatial memory

#### Functional Properties

**Scene Selectivity:**
- Responds more to scenes than objects or faces
- Indoor, outdoor, natural, urban

**Spatial Layout:**
- Geometry of space
- Not individual objects

**Navigation:**
- Environmental landmarks
- Spatial context

### Perirhinal Cortex

**Location:** Along rhinal sulcus, lateral to entorhinal

**Brodmann Areas:** 35, 36

**Function:** Object recognition memory — "Have I seen this before?" Familiarity signals.

#### Connectivity

**From IT:**
- Complex object representations

**To Entorhinal Cortex:**
- Input to hippocampal memory system
- "What" stream to hippocampus

**To/From Amygdala:**
- Emotional significance of objects

**To/From Prefrontal:**
- Object memory retrieval

#### Functional Properties

**Familiarity:**
- Recognition without recollection
- "I know this but don't remember when/where"

**Object Memory:**
- Learns novel objects
- Distinguishes similar objects

### Entorhinal Cortex

**See Limbic Structures document for detailed connectivity**

**Function:** Gateway to hippocampus; contains grid cells for spatial navigation

## Part 10: Temporal Pole

**Location:** Anterior tip of temporal lobe

**Brodmann Area:** 38

**Function:** Semantic hub — abstract concepts, social knowledge, person identity, emotional processing. Highest level of temporal lobe hierarchy.

### Within Temporal Lobe

**From STG, MTG, ITG:**
- All temporal streams converge
- Auditory, visual, semantic

**From STS:**
- Social information

**From Amygdala:**
- Emotional information

**To All Temporal Areas (Feedback):**
- Layer V/VI → lower areas Layer I

### With Frontal Lobe

**To/From Orbitofrontal Cortex:**
- **Strong bidirectional**
- Emotional significance
- Social decision-making

**To/From Ventromedial PFC:**
- Self, social cognition

**To/From Inferior Frontal:**
- Semantic retrieval

### With Limbic Structures

**To/From Amygdala:**
- **Bidirectional, strong**
- Emotional semantics
- Social-emotional

**To/From Hippocampus:**
- Via entorhinal
- Semantic memory

### With Parietal Lobe

**To/From Angular Gyrus:**
- Semantic integration

### Functional Properties

**Semantic Hub:**
- Abstract concepts
- Convergence of multimodal information

**Social Knowledge:**
- Person identity (name, biography)
- Social concepts
- Famous faces

**Emotional Semantics:**
- Emotional concepts
- Social emotions (guilt, embarrassment)

**Semantic Dementia:**
- Progressive atrophy of temporal poles
- Loss of word meanings
- Loss of person knowledge
- Objects become unfamiliar

## Part 12: Language Network Summary (Left Hemisphere)

### Dorsal Language Stream

**Path:** Auditory cortex → Posterior STG → Supramarginal gyrus → Broca's area (via arcuate fasciculus)

**Function:** Phonological processing, speech production, repetition

**White Matter:** Arcuate fasciculus, superior longitudinal fasciculus

### Ventral Language Stream

**Path:** Auditory cortex → Anterior STG → MTG → Temporal pole → Ventral frontal (via uncinate, IFOF)

**Function:** Semantic processing, comprehension, word meanings

**White Matter:** Uncinate fasciculus, inferior fronto-occipital fasciculus (IFOF)

### Key Language Areas in Temporal Lobe

| Area | Function | Deficit |
|------|----------|---------|
| Posterior STG (Wernicke's) | Phonological, word recognition | Wernicke's aphasia |
| MTG | Semantic processing | Semantic deficits |
| Anterior STG | Sentence processing | Comprehension deficits |
| Temporal pole | Semantic hub | Semantic dementia |

## Part 13: Ventral Visual Stream Summary

### Hierarchy

```
V1 → V2 → V4 → TEO (posterior IT) → TE (anterior IT)
                  ↓                      ↓
              Fusiform ←→ Temporal pole
                  ↓
              FFA, VWFA
```

### Transformations Along Hierarchy

| Level | Receptive Field | Selectivity | Invariance |
|-------|-----------------|-------------|------------|
| V1 | Small | Orientation, edges | None |
| V4 | Medium | Color, curves | Some position |
| TEO | Large | Complex features | More position |
| TE | Very large | Objects, categories | Position, size, view |
| FFA/VWFA | Large | Faces/Words | High invariance |

### Feedback Throughout

Every level sends feedback (Layer V/VI → Layer I) to earlier levels:
- Predictive coding
- Object-based attention
- Top-down recognition
