# Region-Centric Communication Map: Cerebrum — Cerebral Cortex — Parietal Lobe

This document provides a complete communication map for every major region of the parietal lobe. For each region, it specifies what that area communicates with at every level: within the parietal lobe, with other cortical lobes, with subcortical structures, and with brainstem structures — all with layer-level connectivity where applicable.

---

## Foundational Reference

**Before reading this document, ensure familiarity with the Canonical Cortical Microcircuit document** (`cerebrum-Cerebral-Cortex-Canonical-Microcircuit.md`), which describes:
- The six cortical layers and their cell types
- Feedforward connections: Layer III → Layer IV of target
- Feedback connections: Layer V/VI → Layer I of target (avoiding Layer IV)
- Layer IV as the input layer; Layer V as subcortical output; Layer VI as thalamic feedback
- Granular vs agranular cortex types

The parietal lobe contains both **granular cortex** (primary somatosensory cortex with thick Layer IV) and **dysgranular cortex** (posterior parietal association areas with moderate Layer IV).

## Part 1: Overview of the Parietal Lobe

### Anatomical Location

The parietal lobe occupies the superior-posterior portion of the cerebral hemisphere. Key boundaries:

**Anterior:** Central sulcus (separates from frontal lobe)
**Posterior:** Parieto-occipital sulcus (separates from occipital lobe on medial surface)
**Inferior:** Lateral (Sylvian) fissure and an imaginary line to parieto-occipital sulcus (separates from temporal lobe)
**Medial:** Extends onto medial surface between cingulate sulcus and parieto-occipital sulcus

### Major Subdivisions

**Anterior Parietal Cortex:**
- Primary somatosensory cortex (S1) — postcentral gyrus
- Secondary somatosensory cortex (S2) — parietal operculum

**Posterior Parietal Cortex (PPC):**
- Superior parietal lobule (SPL) — areas 5, 7
- Inferior parietal lobule (IPL) — areas 39 (angular gyrus), 40 (supramarginal gyrus)
- Intraparietal sulcus (IPS) — contains multiple specialized areas

**Medial Parietal Cortex:**
- Precuneus — medial surface, between paracentral lobule and parieto-occipital sulcus

### Functional Overview

The parietal lobe serves three major functions:

**Somatosensory Processing:** Receives and processes touch, proprioception, pain, temperature from the body

**Spatial Processing:** Represents space, attention, and spatial relationships; "where" things are

**Sensorimotor Integration:** Transforms sensory information into motor coordinates for action; "how" to interact with objects

The parietal lobe is a critical node of the **dorsal visual stream** — it receives visual motion and spatial information from occipital cortex and transforms it for action planning.

## Part 2: Primary Somatosensory Cortex (S1)

**Location:** Postcentral gyrus, immediately posterior to the central sulcus

**Brodmann Areas:** 3a, 3b, 1, 2 (from anterior to posterior)

**Function:** First cortical stage of somatosensory processing. Receives touch, proprioception, pain, and temperature information from the body via thalamus. Organized as a somatotopic body map (sensory homunculus).

### Somatotopic Organization

S1 contains a complete map of the body surface (sensory homunculus):
- **Medial (top of brain):** Foot, leg, trunk
- **Lateral (side of brain):** Hand, face, tongue
- **Cortical magnification:** Hand and face have disproportionately large representation (high receptor density, fine discrimination)

### Subregions

**Area 3a (Deepest, in central sulcus):**
- Receives primarily **proprioceptive** input
- Muscle spindle, joint receptor information
- From VPL thalamus (which receives from dorsal column nuclei and Clarke's column)

**Area 3b (Posterior bank of central sulcus):**
- PRIMARY cutaneous (skin) area
- Fine touch, texture
- Receives densest thalamic input
- Considered the "true" S1 by some

**Area 1 (Crown of postcentral gyrus):**
- Texture processing
- Receives from 3b
- Higher-order touch

**Area 2 (Posterior postcentral gyrus):**
- Size, shape processing
- Integrates touch with proprioception
- Object manipulation information

### Laminar Organization

S1 is **granular cortex** with prominent Layer IV (receives heavy thalamic input).

**Layer IV:** Thick; receives VPL/VPM thalamic input; organized into "barrels" in rodent whisker cortex (similar organization in primates for fingertips)

### Within Parietal Lobe

**To S2 (Secondary Somatosensory Cortex):**
- **Layer III → S2 Layer IV**
- Feedforward
- All S1 subregions project to S2

**To Posterior Parietal Cortex (Area 5):**
- **Layer III → Area 5 Layer IV**
- Proprioceptive and touch for action
- Especially from areas 3a and 2

**To Area 7:**
- Via Area 5
- Sensorimotor integration

**Between S1 Subregions:**
- 3b → 1 → 2 (hierarchical)
- 3a → 2 (proprioception to shape)
- Layer III → Layer IV connections

### With Frontal Lobe

**To Primary Motor Cortex (M1):**
- **Layer III → M1 Layer IV** (though M1 has thin Layer IV)
- Somatosensory feedback for motor control
- Especially from areas 3a (proprioception) and 2

**To Premotor Cortex:**
- Somatosensory information for motor planning

**From M1 (Feedback):**
- M1 Layer V/VI → S1 Layer I
- Motor-related modulation
- Efference copy signals

### With Thalamus

**From VPL (Ventral Posterolateral Nucleus):**
- MAJOR INPUT — driver input
- VPL → S1 Layer IV (primarily 3a, 3b)
- Body somatosensation (below head)
- Medial lemniscus pathway: Fine touch, proprioception
- Spinothalamic pathway: Pain, temperature

**From VPM (Ventral Posteromedial Nucleus):**
- VPM → S1 Layer IV (face area)
- Face somatosensation
- Trigeminal pathway

**To VPL/VPM (Feedback):**
- **Layer VI → VPL/VPM**
- Modulates thalamic relay
- Attention effects

**To/From Pulvinar:**
- Bidirectional
- Higher-order processing

**To TRN:**
- Layer VI axon collaterals → TRN
- Gating of somatosensory input

### With Subcortical Structures

**To Striatum (Putamen):**
- **Layer V → Putamen**
- Somatosensory input to motor basal ganglia
- Somatotopically organized

**To Superior Colliculus:**
- **Layer V → Superior colliculus**
- Orienting to tactile stimuli

**To Pontine Nuclei:**
- **Layer V → Pontine nuclei → Cerebellum**
- Somatosensory for cerebellar processing

**To Claustrum:**
- Layer V → Claustrum

### From Brainstem (Neuromodulation)

**From Locus Coeruleus:**
- Noradrenergic to all layers
- Enhances signal-to-noise
- Attention effects

**From Raphe Nuclei:**
- Serotonergic modulation

**From Basal Forebrain:**
- Cholinergic (from nucleus basalis)
- Plasticity, attention

### Layer-by-Layer Summary for S1

| Layer | Receives From | Sends To | Function |
|-------|--------------|----------|----------|
| I | S2, PPC feedback; pulvinar | Local modulation | Top-down modulation |
| II/III | Layer IV | S2, PPC (Layer IV); areas 1, 2 | Processing; feedforward output |
| IV | VPL/VPM thalamus; area 3b (for 1, 2) | Layers II/III, V, VI | Input reception |
| V | Layers II/III, IV | Putamen, superior colliculus, pons; M1 | Subcortical output |
| VI | Layers IV, V | VPL/VPM, TRN | Thalamic feedback |

## Part 3: Secondary Somatosensory Cortex (S2)

**Location:** Parietal operculum (upper bank of lateral sulcus); posterior to primary somatosensory cortex

**Brodmann Area:** 43 and adjacent areas

**Function:** Higher-order somatosensory processing. Integrates bilateral body information. Processes tactile object recognition, texture discrimination, pain perception. Less strictly somatotopic than S1.

### Within Parietal Lobe

**From S1:**
- S1 Layer III → S2 Layer IV
- All S1 subregions project to S2
- Feedforward

**To S1 (Feedback):**
- **Layer V/VI → S1 Layer I**
- Top-down modulation

**To Posterior Parietal Cortex:**
- **Layer III → PPC Layer IV**
- Tactile for action

**To Insula:**
- **Layer III → Posterior insula Layer IV**
- Pain, interoception

### With Temporal Lobe

**To Temporal Cortex:**
- Tactile object recognition pathway
- Analogous to ventral visual stream for touch

### With Frontal Lobe

**To Premotor Cortex:**
- Tactile information for motor planning

**To Prefrontal Cortex:**
- Tactile working memory

### With Thalamus

**From VPL/VPM:**
- Direct thalamic input (parallel to S1)
- Layer IV

**From Pulvinar:**
- Attention modulation

**To Thalamus:**
- Layer VI feedback

### Functional Properties

**Bilateral Representation:**
- Unlike S1, S2 receives input from BOTH sides of body
- Via corpus callosum from contralateral S1

**Tactile Object Recognition:**
- "What" pathway for touch
- Shape, texture integration

**Pain Processing:**
- Receives nociceptive information
- Projects to insula, cingulate

**Attention Effects:**
- Strong attentional modulation
- More than S1

## Part 4: Superior Parietal Lobule (SPL) — Area 5

**Location:** Superior parietal lobule, posterior to postcentral gyrus

**Brodmann Area:** 5

**Function:** Somatosensory association cortex. Integrates proprioceptive and tactile information for body schema and reaching. Critical for knowing where your body is in space.

### Within Parietal Lobe

**From S1:**
- S1 Layer III → Area 5 Layer IV
- Especially from areas 3a (proprioception) and 2 (shape)

**To Area 7:**
- **Layer III → Area 7 Layer IV**
- Integration with visual-spatial

**To Intraparietal Sulcus (MIP, AIP):**
- Reaching, grasping planning

**To S1 (Feedback):**
- Layer V/VI → S1 Layer I

### With Frontal Lobe

**To Premotor Cortex:**
- **Layer III → Premotor Layer IV**
- Body information for motor planning

**To Primary Motor Cortex:**
- Proprioceptive feedback
- Reaching coordination

**To SMA:**
- Body representation for movement sequences

**From Prefrontal Cortex:**
- Goal-directed modulation

### With Occipital Lobe

**From Visual Areas:**
- Minor input from dorsal visual stream
- Integrates visual and proprioceptive

### With Thalamus

**From Pulvinar:**
- Higher-order thalamic input

**From VPL (Minor):**
- Some direct proprioceptive

**To Thalamus:**
- Layer VI feedback

### With Subcortical Structures

**To Striatum (Caudate):**
- **Layer V → Caudate**
- Spatial information for action selection

**To Pontine Nuclei:**
- **Layer V → Pontine nuclei → Cerebellum**
- Body information for cerebellar coordination

**To Superior Colliculus:**
- Layer V → SC
- Body-related orienting

### Functional Properties

**Body Schema:**
- Internal model of body position
- Knows where limbs are without looking

**Reaching Preparation:**
- Transforms body coordinates for reaching
- "Where is my arm relative to target?"

**Proprioceptive Integration:**
- Combines muscle, joint, skin signals
- Integrated body representation

**Deficits (Lesions):**
- Astereognosis (inability to identify objects by touch)
- Impaired tactile discrimination
- Disturbed body schema

## Part 5: Superior Parietal Lobule (SPL) — Area 7

**Location:** Posterior part of superior parietal lobule

**Brodmann Area:** 7 (parts — 7a, 7b)

**Function:** Visuomotor integration. Combines visual and somatosensory information for spatial awareness and action planning. Contains neurons responsive to visual, tactile, AND proprioceptive stimuli.

### Subdivisions

**Area 7a:** Visual-spatial; dorsal, more visual input
**Area 7b:** Somatosensory-motor; more tactile/proprioceptive

### Within Parietal Lobe

**From Area 5:**
- Area 5 Layer III → Area 7 Layer IV
- Somatosensory information

**From Intraparietal Sulcus (LIP, VIP, MIP):**
- Visual-spatial information

**To/From Inferior Parietal Lobule:**
- Integration across PPC

**To Area 5 (Feedback):**
- Layer V/VI → Area 5 Layer I

### With Occipital Lobe

**From Visual Areas (MT, MST, V3A):**
- **Visual Layer III → Area 7 Layer IV**
- Motion, spatial information
- Dorsal stream input

**To Visual Areas (Feedback):**
- **Layer V/VI → Visual areas Layer I**
- Attention, spatial selection

### With Frontal Lobe

**To Premotor Cortex:**
- **Layer III → Premotor Layer IV**
- Spatial coordinates for action

**To Frontal Eye Fields (FEF):**
- Spatial information for saccades

**To Prefrontal Cortex (DLPFC):**
- Spatial working memory
- Spatial reasoning

**From Prefrontal (Feedback):**
- Goal-directed attention
- Task demands

### With Temporal Lobe

**To/From Superior Temporal Sulcus (STS):**
- Biological motion
- Social attention

### With Thalamus

**From Pulvinar:**
- **Pulvinar → Area 7 Layer IV and I**
- Visual attention
- Multimodal integration

**To Pulvinar:**
- **Layer V → Pulvinar**
- Transthalamic pathways

### With Subcortical Structures

**To Superior Colliculus:**
- **Layer V → SC**
- Spatial orienting
- Attention shifts

**To Striatum:**
- **Layer V → Caudate**
- Spatial action selection

**To Pontine Nuclei:**
- **Layer V → Pons → Cerebellum**
- Spatial coordination

### Functional Properties

**Spatial Representation:**
- Multiple coordinate frames
- Eye-centered, head-centered, body-centered

**Visuomotor:**
- Visually-guided reaching
- Coordinate transformation

**Attention:**
- Spatial attention shifts
- Salience representation

**Multimodal:**
- Combines visual, auditory, tactile spatial information

## Part 6: Intraparietal Sulcus (IPS) — Overview

**Location:** Major sulcus running anterior-posterior through the posterior parietal cortex, separating superior and inferior parietal lobules

**Function:** Contains multiple specialized areas for different aspects of spatial processing and sensorimotor transformation. Critical hub of the dorsal stream.

### IPS Areas

**LIP (Lateral Intraparietal):** Eye movements, spatial attention
**VIP (Ventral Intraparietal):** Multisensory space, self-motion
**MIP (Medial Intraparietal):** Reaching
**AIP (Anterior Intraparietal):** Grasping, object manipulation
**CIP (Caudal Intraparietal):** 3D object orientation, surfaces

---

## Part 7: Lateral Intraparietal Area (LIP)

**Location:** Lateral bank of intraparietal sulcus

**Function:** Saccadic eye movement planning and spatial attention. Represents salience and priority of locations in the visual field. Critical for deciding where to look next.

### Within Parietal Lobe

**From Area 7:**
- Spatial information

**From VIP:**
- Multisensory spatial

**To/From Other IPS Areas:**
- Integrated spatial processing

### With Occipital Lobe

**From MT, MST:**
- **MT/MST Layer III → LIP Layer IV**
- Motion, spatial information

**From V3A:**
- Visual spatial

**To Visual Areas (Feedback):**
- **Layer V/VI → Visual areas Layer I**
- Spatial attention modulation

### With Frontal Lobe

**To/From FEF (Frontal Eye Fields):**
- **Bidirectional**
- LIP Layer III → FEF Layer IV (feedforward for saccades)
- FEF Layer V/VI → LIP Layer I (feedback)
- Saccade planning coordination

**To/From DLPFC:**
- Spatial working memory
- Goal-directed attention

**To/From SEF:**
- Saccade sequences

### With Thalamus

**From Pulvinar:**
- **Pulvinar → LIP Layer IV, I**
- Attention modulation
- Visual salience

**To Pulvinar:**
- **Layer V → Pulvinar**

**From MD Thalamus:**
- Prefrontal-related

### With Subcortical Structures

**To Superior Colliculus:**
- **Layer V → SC**
- MAJOR projection
- Saccade commands
- Spatial attention

**To Striatum (Caudate):**
- **Layer V → Caudate (body and tail)**
- Oculomotor loop of basal ganglia

### Functional Properties

**Priority/Salience Map:**
- Represents importance of locations
- Integrates bottom-up salience and top-down goals

**Saccade Planning:**
- Activity predicts upcoming saccade
- Accumulates evidence for eye movement

**Spatial Attention:**
- Attention shifts precede saccades
- Covert attention (without eye movement)

**Decision-Making:**
- Evidence accumulation
- Perceptual decisions about motion direction (classical paradigm)

**Neurons Encode:**
- Saccade target location
- Attention location
- Reward expectation
- Elapsed time (temporal integration)

## Part 8: Ventral Intraparietal Area (VIP)

**Location:** Fundus (floor) of intraparietal sulcus

**Function:** Multisensory representation of near-personal space. Integrates visual, tactile, auditory, and vestibular information. Critical for representing space near the face and body, and for perceiving self-motion.

### Within Parietal Lobe

**From Area 7:**
- Spatial information

**To/From LIP:**
- Spatial attention coordination

**To/From MIP:**
- Reaching space

### With Occipital Lobe

**From MT, MST:**
- **Layer III → VIP Layer IV**
- Motion, especially optic flow
- Heading direction

**From V3A:**
- Motion, depth

### With Frontal Lobe

**To Premotor Cortex (PMv — Ventral Premotor):**
- **Layer III → PMv Layer IV**
- Near-space representation
- Defensive movements

**To FEF:**
- Spatial information

### With Temporal/Insular Cortex

**To/From PIVC (Parietal Insular Vestibular Cortex):**
- Vestibular integration
- Self-motion perception

### With Thalamus

**From Pulvinar:**
- Multisensory spatial

**From VPL:**
- Tactile (head, face)

**From Vestibular Thalamus:**
- Vestibular signals

### Functional Properties

**Multisensory Space:**
- Visual receptive fields near body
- Corresponding tactile receptive fields on body surface
- "Visual-tactile" neurons

**Personal Space:**
- Represents space within arm's reach
- Defensive responses (looming stimuli)

**Self-Motion:**
- Optic flow processing
- Vestibular-visual integration
- Heading direction

**Head-Centered Coordinates:**
- Many neurons in head-centered frame
- Stable representation during eye movements

## Part 9: Medial Intraparietal Area (MIP) / Parietal Reach Region (PRR)

**Location:** Medial bank of intraparietal sulcus

**Other Name:** Parietal Reach Region (PRR)

**Function:** Planning reaching movements. Transforms visual target location into reach coordinates. Critical for visually-guided reaching.

### Within Parietal Lobe

**From Area 5:**
- Proprioceptive (arm position)

**From Area 7:**
- Visual-spatial

**To/From LIP:**
- Coordinate eye and reach

**To/From AIP:**
- Reach and grasp coordination

### With Occipital Lobe

**From Dorsal Visual Stream (V3A, V6):**
- **Layer III → MIP Layer IV**
- Visual target location
- Depth information

### With Frontal Lobe

**To Premotor Cortex (PMd — Dorsal Premotor):**
- **Layer III → PMd Layer IV**
- MAJOR projection
- Reach planning

**To Primary Motor Cortex:**
- Reach execution

**To SMA:**
- Movement sequences

**From Prefrontal Cortex:**
- Goal-directed reaching

### With Thalamus

**From Pulvinar:**
- Visual spatial

**From VL Thalamus:**
- Cerebellar influence

### With Subcortical Structures

**To Pontine Nuclei:**
- **Layer V → Pons → Cerebellum**
- Reach coordination

**To Superior Colliculus:**
- Reach-related orienting

### Functional Properties

**Reach Planning:**
- Activity during delay before reaching
- Encodes intended reach direction

**Coordinate Transformation:**
- Visual (eye-centered) → arm-centered
- Accounts for eye, head, arm position

**Effector Specificity:**
- Some neurons specific to reaching (vs. saccades)
- Distinct from LIP (saccades)

**Optic Ataxia:**
- Damage causes misreaching to visual targets
- Reach errors especially in peripheral vision

## Part 10: Anterior Intraparietal Area (AIP)

**Location:** Anterior portion of intraparietal sulcus, near postcentral sulcus

**Function:** Grasping and object manipulation. Represents 3D shape for hand pre-shaping. Critical for visually-guided grasping.

### Within Parietal Lobe

**From S1 (Area 2):**
- Tactile shape information

**From Area 5:**
- Hand proprioception

**To/From MIP:**
- Reach-grasp coordination

### With Occipital Lobe

**From Visual Areas (CIP, V3A, LOC):**
- **Layer III → AIP Layer IV**
- 3D object shape
- Graspable features

**From Ventral Stream (IT):**
- Object identity (what to grasp)

### With Frontal Lobe

**To Ventral Premotor Cortex (PMv, F5):**
- **Layer III → PMv Layer IV**
- MAJOR projection
- Grasp planning
- AIP-F5 circuit = core grasp network

**To Primary Motor Cortex (Hand Area):**
- Grasp execution

**From Prefrontal Cortex:**
- Object selection
- Task goals

### With Thalamus

**From Pulvinar:**
- Visual shape

**From VL:**
- Motor-related

### Functional Properties

**Grasp Planning:**
- Activity during grasp preparation
- Encodes grip type (precision, power, etc.)

**3D Shape for Action:**
- Size, shape, orientation
- "Affordances" — what actions object allows

**Hand Pre-shaping:**
- Hand conforms to object before contact
- Requires AIP-PMv circuit

**Visuomotor Neurons:**
- Respond to vision of graspable objects
- Also respond during grasping (visual + motor)

**Mirror-like Properties:**
- Some neurons respond to observed grasping
- Part of action understanding system

## Part 11: Caudal Intraparietal Area (CIP)

**Location:** Caudal (posterior) part of intraparietal sulcus

**Function:** Processes 3D surface orientation and object structure. Represents slant, tilt, and depth of surfaces. Provides shape information to AIP for grasping.

### With Occipital Lobe

**From V3, V3A:**
- **Layer III → CIP Layer IV**
- Disparity (stereoscopic depth)
- Surface orientation

### Within Parietal Lobe

**To AIP:**
- 3D shape for grasping

**To MIP:**
- Surface information for reaching

### Functional Properties

**Surface Orientation:**
- Slant (tilt away from frontoparallel)
- Tilt (rotation of surface)

**Stereoscopic Depth:**
- Binocular disparity processing

**3D Object Structure:**
- Combines surfaces into object shapes

## Part 12: Inferior Parietal Lobule (IPL) — Supramarginal Gyrus (Area 40)

**Location:** Anterior part of inferior parietal lobule, surrounding the posterior end of the lateral (Sylvian) fissure

**Brodmann Area:** 40

**Function:** Phonological processing (language sounds), tool use, body awareness, and tactile attention. Part of language network in left hemisphere.

### Within Parietal Lobe

**From S1, S2:**
- Somatosensory information

**From Superior Parietal Lobule:**
- Spatial, proprioceptive

**To/From Angular Gyrus:**
- Language integration

### With Temporal Lobe

**To/From Wernicke's Area (Left Hemisphere):**
- Phonological processing
- Sound-meaning mapping

**To/From Superior Temporal Gyrus:**
- **Layer III ↔ STG**
- Auditory-phonological

**From Auditory Cortex:**
- Sound information

### With Frontal Lobe

**To/From Broca's Area (Left Hemisphere):**
- **Bidirectional via arcuate fasciculus**
- Phonological loop
- Speech production planning

**To Premotor Cortex:**
- Tool use, action planning

**From Prefrontal Cortex:**
- Task demands, language tasks

### With Thalamus

**From Pulvinar:**
- Attention modulation

**From VPL:**
- Somatosensory

### Functional Properties

**Phonological Processing (Left):**
- Sound structure of words
- Phonological working memory
- Rhyming, sound manipulation

**Tool Use:**
- Knowledge of tool actions
- Praxis (skilled movements)

**Body Awareness:**
- Sense of body ownership
- Body schema

**Tactile Attention:**
- Attending to touch locations

**Deficits (Lesions):**
- Left: Conduction aphasia (impaired repetition), phonological deficits
- Right: Anosognosia (unawareness of deficit), neglect components

## Part 13: Inferior Parietal Lobule (IPL) — Angular Gyrus (Area 39)

**Location:** Posterior part of inferior parietal lobule, surrounding the posterior end of superior temporal sulcus

**Brodmann Area:** 39

**Function:** Semantic processing, reading, mathematical cognition, episodic memory retrieval, and conceptual integration. Major association hub.

### Within Parietal Lobe

**From Supramarginal Gyrus:**
- Phonological information

**From Superior Parietal Lobule:**
- Spatial information

**To/From Precuneus:**
- Episodic memory, self-reference

### With Temporal Lobe

**To/From Middle Temporal Gyrus:**
- **Bidirectional**
- Semantic processing
- Word meanings

**To/From Temporal Pole:**
- Conceptual, semantic

**From Visual Areas (Ventral Stream):**
- Object information

### With Frontal Lobe

**To/From Prefrontal Cortex (DLPFC, mPFC):**
- **Bidirectional**
- Executive control of semantics
- Memory retrieval

**To/From Broca's Area:**
- Semantic aspects of language

### With Occipital Lobe

**From Visual Cortex:**
- Visual word form
- Reading pathway

### With Thalamus

**From Pulvinar:**
- Attention

**From Lateral Posterior Nucleus:**
- Association

### With Limbic Structures

**To/From Hippocampus (via Parahippocampal):**
- Episodic memory retrieval

**To/From Posterior Cingulate:**
- Self-reference, memory

### Default Mode Network

Angular gyrus is a KEY HUB of the default mode network (DMN):
- Active during rest, self-reflection
- Semantic, autobiographical processing
- Deactivates during attention-demanding tasks

### Functional Properties

**Semantic Processing:**
- Word meanings
- Conceptual knowledge
- Category relationships

**Reading:**
- Visual word form processing
- Phonological-orthographic mapping
- Left angular gyrus critical for reading

**Mathematical Cognition:**
- Number processing
- Arithmetic facts
- Especially left angular gyrus

**Episodic Memory:**
- Retrieval success
- Recollection
- "Remembering" vs. "knowing"

**Spatial Attention:**
- Right hemisphere especially
- Reorienting attention

**Deficits (Lesions):**
- Left: Gerstmann syndrome (agraphia, acalculia, finger agnosia, left-right confusion)
- Alexia (reading impairment)
- Semantic deficits

## Part 14: Precuneus

**Location:** Medial surface of parietal lobe, between paracentral lobule and parieto-occipital sulcus; posterior to marginal sulcus

**Brodmann Area:** 7m (medial), 31

**Function:** Self-referential processing, episodic memory retrieval, visuospatial imagery, and consciousness. Highest metabolic rate of any cortical region. Key hub of default mode network.

### Within Parietal Lobe

**To/From Posterior Parietal Cortex:**
- Spatial imagery

**To/From Angular Gyrus:**
- Semantic memory, self-reference

### With Temporal Lobe

**To/From Hippocampal Formation:**
- **Via retrosplenial cortex, parahippocampal**
- Episodic memory
- Spatial memory

**To/From Temporal Pole:**
- Autobiographical memory

### With Frontal Lobe

**To/From Medial Prefrontal Cortex (mPFC):**
- **Strong connection**
- Self-referential processing
- DMN core

**To/From DLPFC:**
- Working memory imagery

**To/From FEF:**
- Mental imagery of space

### With Occipital Lobe

**To/From Visual Cortex:**
- Visual imagery
- Mental rotation

**To/From Retrosplenial Cortex:**
- Spatial navigation
- Scene processing

### With Cingulate Cortex

**To/From Posterior Cingulate Cortex (PCC):**
- **Very strong connection**
- Core of DMN
- Self-reference

### With Thalamus

**From Pulvinar:**
- Attention

**From Lateral Posterior Nucleus:**
- Association

**From Anterior Thalamus:**
- Memory (Papez circuit related)

### Default Mode Network Hub

Precuneus is central to DMN:
- Among highest connectivity in brain
- Highly metabolically active even at rest
- Deactivates during external attention tasks

### Functional Properties

**Self-Referential Processing:**
- Thinking about self
- First-person perspective

**Episodic Memory:**
- Memory retrieval
- Autobiographical memory
- Mental time travel

**Visuospatial Imagery:**
- Mental rotation
- Spatial perspective taking
- Navigation imagery

**Consciousness:**
- Awareness
- Among first regions to deactivate in anesthesia
- Reduced activity in vegetative states

**Theory of Mind:**
- Thinking about others' thoughts
- Perspective taking
