# Region-Centric Communication Map: Cerebrum — Cerebral Cortex — Occipital Lobe

This document provides a complete communication map for every major region of the occipital lobe. For each region, it specifies what that area communicates with at every level: within the occipital lobe, with other cortical lobes, with subcortical structures (especially thalamus), and with brainstem structures — all with layer-level connectivity where applicable.

---

## Foundational Reference

**Before reading this document, ensure familiarity with the Canonical Cortical Microcircuit document** (`cerebrum-Cerebral-Cortex-Canonical-Microcircuit.md`), which describes:
- The six cortical layers and their cell types
- Feedforward connections: Layer III → Layer IV of target
- Feedback connections: Layer V/VI → Layer I of target (avoiding Layer IV)
- Layer IV as the input layer; Layer V as subcortical output; Layer VI as thalamic feedback
- Granular vs agranular cortex types

The occipital lobe is **highly granular cortex** — thick Layer IV optimized for receiving visual input from the thalamus.

## Part 1: Overview of the Occipital Lobe

### Anatomical Location

The occipital lobe occupies the posterior pole of the brain, behind the parietal and temporal lobes. It is the smallest of the four main lobes. Key landmarks:

**Calcarine Sulcus:** Major sulcus on the medial surface; primary visual cortex (V1) wraps around it

**Parieto-Occipital Sulcus:** Separates occipital lobe from parietal lobe on the medial surface

**Lateral Occipital Sulcus:** Separates superior and inferior occipital gyri

### Functional Overview

The occipital lobe is **entirely dedicated to visual processing**. It contains:
- Primary visual cortex (V1): First cortical stage of visual processing
- Extrastriate visual areas (V2, V3, V4, V5/MT, etc.): Higher-order visual processing

### Visual Processing Streams

Visual information flows from V1 into two major processing streams:

**Dorsal Stream ("Where/How" pathway):**
- V1 → V2 → V3 → V3A → MT (V5) → MST → Posterior Parietal Cortex
- Processes motion, spatial location, action guidance
- "Vision for action"

**Ventral Stream ("What" pathway):**
- V1 → V2 → V4 → Inferior Temporal Cortex (IT)
- Processes object identity, color, form
- "Vision for perception"

### Retinotopic Organization

All early visual areas maintain **retinotopic maps** — neighboring points in the visual field are represented by neighboring neurons in the cortex:
- **Fovea (central vision):** Overrepresented; occupies large cortical area (cortical magnification)
- **Periphery:** Less cortical area
- **Contralateral representation:** Left visual field → right occipital cortex

## Part 2: Primary Visual Cortex (V1) — Striate Cortex

**Location:** Surrounds the calcarine sulcus on the medial surface of the occipital lobe; extends slightly onto the lateral surface at the occipital pole.

**Brodmann Area:** 17

**Other Names:** Striate cortex (due to visible "stria of Gennari" — myelinated fibers in Layer IVB), calcarine cortex

**Function:** First cortical stage of visual processing. Extracts basic visual features: orientation, spatial frequency, direction of motion, binocular disparity, color.

### Unique Laminar Features

V1 has the most elaborate Layer IV of any cortical region, subdivided into:

**Layer IVA:** Receives some input from Layer III of other areas

**Layer IVB:** Heavily myelinated (stria of Gennari visible to naked eye)
- Contains large pyramidal and stellate cells
- Projects directly to MT (V5) for motion processing
- Receives magnocellular stream information

**Layer IVCα:** Receives **magnocellular LGN** input
- Large receptive fields
- Motion, low spatial frequency, low contrast sensitivity
- Projects to Layer IVB → dorsal stream

**Layer IVCβ:** Receives **parvocellular LGN** input
- Small receptive fields
- Color, high spatial frequency, fine detail
- Projects to Layers II/III → ventral stream

**Layers II/III:** Contain **blobs** and **interblobs**
- Blobs: Color processing (cytochrome oxidase rich)
- Interblobs: Orientation and form

### Within Occipital Lobe

**To V2:**
- **Layer III (interblobs) → V2 Layer IV (pale stripes):** Orientation, form
- **Layer III (blobs) → V2 Layer IV (thin stripes):** Color
- **Layer IVB → V2 Layer IV (thick stripes):** Motion, depth
- Feedforward connections

**To V3:**
- Layer III → V3 Layer IV
- Some direct projection, especially from peripheral representation

**To MT (V5):**
- **Layer IVB → MT Layer IV:** DIRECT projection for motion
- Bypasses V2 for fast motion processing
- Also Layer III contribution

**From V2 (Feedback):**
- V2 Layer V/VI → V1 Layer I, II/III (avoids Layer IV)
- Modulatory, contextual

**From V3, V4, MT (Feedback):**
- All send feedback to V1 Layer I
- Top-down predictions and context

### With Thalamus

**From Lateral Geniculate Nucleus (LGN):**
- MAJOR INPUT — driver input
- LGN → V1 Layer IV
  - Magnocellular layers (1-2) → Layer IVCα
  - Parvocellular layers (3-6) → Layer IVCβ
  - Koniocellular layers → Layers II/III (blobs)
- Retinotopically organized

**To LGN (Feedback):**
- **Layer VI → LGN:** MASSIVE corticothalamic feedback
- 10x more V1→LGN synapses than LGN→V1
- Modulates LGN processing
- Sharpens receptive fields
- Attention effects

**To Pulvinar:**
- **Layer V → Pulvinar (inferior):** Transthalamic pathway
- Enables V1 to influence higher visual areas via pulvinar

**From Pulvinar:**
- Pulvinar → V1 Layer I: Modulatory, attention

**To/From TRN (Thalamic Reticular Nucleus):**
- V1 corticothalamic axons send collaterals to TRN (visual sector)
- TRN inhibits LGN
- Creates attentional gating of visual input

### With Other Cortical Lobes

**To Parietal Lobe:**
- Via V2 → V3 → V3A → MT → Posterior parietal cortex
- Dorsal stream for spatial vision and action

**To Temporal Lobe:**
- Via V2 → V4 → Inferior temporal cortex
- Ventral stream for object recognition

**Feedback from Parietal Cortex:**
- Posterior parietal → V1 Layer I
- Attention, spatial selection

**Feedback from Frontal Cortex (FEF):**
- FEF → V1 Layer I (via pulvinar primarily)
- Attentional modulation
- Saccade-related modulation

### With Subcortical Structures

**To Superior Colliculus:**
- **Layer V → Superior colliculus**
- Orienting responses
- Saccadic eye movements

**To Claustrum:**
- Layer V → Claustrum
- Function debated; may coordinate across modalities

**From Basal Forebrain:**
- Cholinergic modulation from nucleus basalis
- Enhances attention-related processing

### From Brainstem

**From Locus Coeruleus:**
- Noradrenergic modulation to all layers
- Arousal, attention, gain modulation

**From Raphe Nuclei:**
- Serotonergic modulation
- State-dependent processing

**From VTA:**
- Dopaminergic (minor in V1)
- Reward-related modulation

### Functional Organization

**Orientation Columns:**
- Neurons preferring similar orientations grouped vertically
- All orientations represented within ~500 μm

**Ocular Dominance Columns:**
- Alternating stripes preferring left vs right eye
- ~500 μm width each

**Blobs and Interblobs:**
- Blobs: Color-selective, located in Layers II/III
- Interblobs: Orientation-selective

**Hypercolumns:**
- ~1 mm² containing full set of orientations and both eyes
- Complete representation of one point in visual space

**Receptive Fields:**
- Simple cells: Orientation-selective, phase-sensitive (Layer IV)
- Complex cells: Orientation-selective, phase-invariant (Layers II/III, V)
- End-stopped cells: Respond to line endings, corners

### Layer-by-Layer Summary for V1

| Layer | Input From | Output To | Function |
|-------|-----------|-----------|----------|
| I | V2, V3, V4, MT, pulvinar (feedback) | Local modulation | Top-down modulation |
| II/III (blobs) | Layer IVCβ, koniocellular | V2 thin stripes | Color processing |
| II/III (interblobs) | Layer IVCβ | V2 pale stripes | Orientation, form |
| IVA | V2 Layer III | Local | Corticocortical input |
| IVB | Layer IVCα | MT, V2 thick stripes | Motion |
| IVCα | LGN magnocellular | Layer IVB, II/III | Motion, low contrast |
| IVCβ | LGN parvocellular | Layers II/III | Detail, color |
| V | Layers II/III, IV | Superior colliculus, pulvinar, claustrum | Subcortical output |
| VI | Layers IV, V | LGN (feedback), TRN | Thalamic modulation |

## Part 3: Secondary Visual Cortex (V2) — Prestriate Cortex

**Location:** Surrounds V1 like a horseshoe; anterior to V1 on both medial and lateral surfaces

**Brodmann Area:** 18

**Function:** Second stage of visual processing. Integrates information from V1; beginning of form, depth, and color processing. Contains illusory contour responses.

### Stripe Organization

V2 has a characteristic stripe pattern (visible with cytochrome oxidase staining):

**Thin Stripes:**
- Receive from V1 blobs (color)
- Process color
- Project to V4 (ventral stream)

**Thick Stripes:**
- Receive from V1 Layer IVB (motion, depth)
- Process motion, stereoscopic depth
- Project to MT (dorsal stream)

**Pale (Interstripe) Stripes:**
- Receive from V1 interblobs (orientation, form)
- Process form and shape
- Project to V4 (ventral stream)

### Within Occipital Lobe

**From V1 (Feedforward):**
- V1 Layer III → V2 Layer IV
- Organized by stripe type (see above)
- Retinotopic correspondence maintained

**To V1 (Feedback):**
- **Layer V/VI → V1 Layer I, II/III**
- Contextual modulation
- Illusory contour signals

**To V3:**
- **Layer III → V3 Layer IV**
- Form processing stream
- Both dorsal and ventral contributions

**To V4:**
- **Layer III (thin + pale stripes) → V4 Layer IV**
- Color and form for ventral stream

**To MT (V5):**
- **Layer III (thick stripes) → MT Layer IV**
- Motion and depth for dorsal stream

**From V3, V4, MT (Feedback):**
- Layer V/VI of higher areas → V2 Layer I
- Top-down modulation

### With Thalamus

**From Pulvinar:**
- Pulvinar → V2 Layers I, IV
- Attention modulation
- Transthalamic input from other visual areas

**To Pulvinar:**
- **Layer V → Pulvinar**
- Transthalamic route to other cortical areas

**From LGN:**
- Minor direct input (koniocellular)

**To LGN:**
- Minor feedback

### With Other Cortical Lobes

**To Parietal Lobe:**
- Via V3, MT to posterior parietal cortex
- Dorsal stream continuation

**To Temporal Lobe:**
- Via V4 to inferior temporal cortex
- Ventral stream continuation

**Feedback from Higher Areas:**
- Parietal, temporal, frontal areas project back
- Layer I termination primarily

### With Subcortical Structures

**To Superior Colliculus:**
- Layer V → Superior colliculus
- Orienting, eye movements

**To Claustrum:**
- Layer V → Claustrum

### Functional Properties

**Illusory Contours:**
- V2 neurons respond to edges not physically present
- Inferred from context (e.g., Kanizsa triangle)

**Border Ownership:**
- V2 neurons signal which side of edge belongs to figure vs ground

**Texture Segmentation:**
- Begin to distinguish figure from background based on texture

### Layer Connectivity Summary for V2

| Connection | Origin Layer | Target | Function |
|------------|--------------|--------|----------|
| From V1 | V1 Layer III | V2 Layer IV | Feedforward input |
| To V1 | V2 Layer V/VI | V1 Layer I | Feedback modulation |
| To V3 | Layer III | V3 Layer IV | Feedforward |
| To V4 | Layer III (thin, pale) | V4 Layer IV | Color, form |
| To MT | Layer III (thick) | MT Layer IV | Motion, depth |
| To pulvinar | Layer V | Pulvinar | Transthalamic |
| To superior colliculus | Layer V | SC | Orienting |

## Part 4: Visual Area V3

**Location:** Anterior to V2; consists of two parts:
- **V3 (dorsal V3):** Upper visual field representation, dorsal surface
- **VP (ventral V3/V3v):** Lower visual field representation, ventral surface

**Brodmann Area:** 19 (part)

**Function:** Processes global form, dynamic form (shape from motion), and some orientation processing. Contributes to both dorsal and ventral streams.

### Within Occipital Lobe

**From V1:**
- V1 Layer III → V3 Layer IV
- Direct input, especially peripheral visual field

**From V2:**
- V2 Layer III → V3 Layer IV
- Main feedforward input
- Both thick and pale stripes contribute

**To V2 (Feedback):**
- Layer V/VI → V2 Layer I

**To V1 (Feedback):**
- Layer V/VI → V1 Layer I

**To V3A:**
- Layer III → V3A Layer IV
- Motion and form integration

**To MT:**
- Layer III → MT Layer IV
- Motion processing

**To V4:**
- Layer III → V4 Layer IV
- Form processing (especially ventral V3)

### With Other Cortical Lobes

**To Posterior Parietal Cortex:**
- Via V3A and MT
- Dorsal stream for spatial vision

**To Temporal Cortex:**
- Via V4
- Ventral stream for form

**Feedback from Parietal:**
- LIP, VIP → V3 Layer I
- Attention, spatial selection

### With Thalamus

**From Pulvinar:**
- Pulvinar → V3 Layer I, IV
- Transthalamic and modulatory

**To Pulvinar:**
- Layer V → Pulvinar

### Functional Properties

**Global Form:**
- Responds to large-scale shape
- Shape from motion

**Orientation Processing:**
- Continues orientation analysis from V1/V2

**Color:**
- Some color responses, especially in ventral V3

## Part 5: Visual Area V3A

**Location:** Dorsal to V3, anterior-lateral; part of dorsal stream

**Function:** Intermediate area between early visual cortex and posterior parietal. Processes motion, depth, and spatial relationships. Important for perceiving self-motion.

### Within Occipital Lobe

**From V3:**
- V3 Layer III → V3A Layer IV

**From V2:**
- Direct input from thick stripes

**From V1:**
- Some direct input

**To MT:**
- Layer III → MT Layer IV

**To MST:**
- Layer III → MST Layer IV

### With Parietal Lobe

**To Posterior Parietal Cortex (Area 7, LIP):**
- Layer III → Parietal Layer IV
- Motion for spatial processing

**From Posterior Parietal (Feedback):**
- Parietal Layer V/VI → V3A Layer I

### With Thalamus

**From/To Pulvinar:**
- Bidirectional connections

### Functional Properties

**Motion Processing:**
- Larger receptive fields than V3
- Global motion

**Stereoscopic Depth:**
- Disparity processing

**Self-Motion:**
- Responds to optic flow patterns

## Part 6: Visual Area V4

**Location:** Lateral and ventral occipital surface; anterior to V2/V3; extends into posterior temporal lobe

**Brodmann Area:** 19 (part)

**Function:** Major hub of the ventral stream. Processes color, intermediate-level form, object features, texture. Attention strongly modulates V4 responses.

### Within Occipital Lobe

**From V1:**
- V1 Layer III → V4 Layer IV
- Some direct input (especially from blobs)

**From V2:**
- V2 Layer III (thin stripes) → V4 Layer IV: Color
- V2 Layer III (pale stripes) → V4 Layer IV: Form

**From V3 (ventral):**
- V3 Layer III → V4 Layer IV

**To V1, V2, V3 (Feedback):**
- Layer V/VI → Lower areas Layer I
- Top-down modulation

### With Temporal Lobe

**To Inferior Temporal Cortex (IT):**
- **Layer III → IT Layer IV**
- Major feedforward projection
- Object features for recognition

**To TEO (Posterior IT):**
- Layer III → TEO Layer IV
- Intermediate form processing

**Feedback from IT:**
- IT Layer V/VI → V4 Layer I
- Object-based attention

### With Parietal Lobe

**To/From Posterior Parietal:**
- Bidirectional but weaker than dorsal stream areas
- Some spatial information exchange

### With Frontal Lobe

**From FEF (Frontal Eye Fields):**
- FEF → V4 Layer I (via pulvinar primarily)
- Attentional modulation
- Strong attention effects in V4

**From Prefrontal Cortex:**
- PFC → V4 (via pulvinar)
- Top-down feature selection

### With Thalamus

**From Pulvinar:**
- Pulvinar → V4 Layers I, IV
- Attention modulation (strong effects)

**To Pulvinar:**
- Layer V → Pulvinar
- Transthalamic route

### With Subcortical Structures

**To Claustrum:**
- Layer V → Claustrum

**From Basal Forebrain:**
- Cholinergic modulation
- Enhances attentional effects

### Functional Properties

**Color Processing:**
- Color-selective neurons
- Color constancy (stable color perception under varying illumination)
- Some neurons tuned to specific hues

**Intermediate Form:**
- Curvature, contour features
- Texture boundaries
- Figure-ground processing

**Attention Effects:**
- V4 shows STRONG attentional modulation
- Attended stimuli: Enhanced responses
- Unattended stimuli in RF: Suppressed responses
- "Biased competition" demonstrated in V4

**Receptive Field Properties:**
- Larger than V1, V2
- Moderate complexity
- Some position invariance beginning

### Layer Connectivity Summary for V4

| Connection | Origin Layer | Target | Function |
|------------|--------------|--------|----------|
| From V2 | V2 Layer III | V4 Layer IV | Feedforward (color, form) |
| To IT | V4 Layer III | IT Layer IV | Object features |
| To V2, V1 | V4 Layer V/VI | V2, V1 Layer I | Feedback |
| From FEF/PFC | Via pulvinar | V4 Layer I | Attention |
| To pulvinar | Layer V | Pulvinar | Transthalamic |

## Part 7: Visual Area V5/MT (Middle Temporal Area)

**Location:** Junction of occipital, temporal, and parietal lobes; in the superior temporal sulcus (ascending limb)

**Other Names:** MT (Middle Temporal); V5 in human nomenclature

**Function:** Specialized for MOTION processing. Computes direction and speed of moving stimuli. Critical for motion perception, smooth pursuit eye movements, and perceiving structure from motion.

### Unique Features

**High Myelination:** MT is heavily myelinated, visible in histology

**Columnar Organization:** Direction columns — neurons preferring similar motion directions grouped together

### Within Occipital Lobe

**From V1:**
- **V1 Layer IVB → MT Layer IV:** DIRECT projection
- Bypasses V2 for fast motion information
- Magnocellular stream
- V1 Layer III also contributes

**From V2:**
- V2 Layer III (thick stripes) → MT Layer IV
- Motion and depth information

**From V3, V3A:**
- Layer III → MT Layer IV

**To V1, V2, V3 (Feedback):**
- Layer V/VI → Lower areas Layer I
- Motion-based predictions

### With Temporal Lobe

**To MST (Medial Superior Temporal):**
- **Layer III → MST Layer IV**
- Major feedforward target
- Optic flow, self-motion

**To FST (Floor of Superior Temporal):**
- Motion and form integration

**Feedback from MST:**
- MST Layer V/VI → MT Layer I

### With Parietal Lobe

**To Posterior Parietal Cortex (LIP, VIP, Area 7a):**
- **Layer III → Parietal Layer IV**
- Motion for action planning
- Saccade planning (to LIP)
- Self-motion, heading (to VIP)

**Feedback from Parietal:**
- Parietal Layer V/VI → MT Layer I
- Attentional modulation

### With Frontal Lobe

**To FEF (Frontal Eye Fields):**
- Motion information for saccade/pursuit planning

**From FEF:**
- Attentional modulation
- Saccade-related signals

**To SEF (Supplementary Eye Fields):**
- Motion for eye movement sequences

### With Thalamus

**From Pulvinar:**
- Pulvinar → MT Layer I, IV
- Attention modulation

**To Pulvinar:**
- Layer V → Pulvinar

**From LGN (Direct):**
- Minor direct input from LGN (koniocellular) bypassing V1
- May support blindsight

### With Subcortical Structures

**To Superior Colliculus:**
- Layer V → Superior colliculus
- Motion for orienting

**From Superior Colliculus:**
- Via pulvinar
- Salience, motion detection

### With Brainstem

**To Pontine Nuclei:**
- Layer V → Pontine nuclei → Cerebellum
- Motion for oculomotor control
- Smooth pursuit pathways

### Functional Properties

**Direction Selectivity:**
- Most MT neurons are direction-selective
- Respond to preferred direction of motion
- Suppressed by opposite direction

**Speed Tuning:**
- Neurons tuned to specific speeds
- Wide range of speed preferences

**Pattern Motion:**
- Some neurons respond to pattern (object) motion
- Not just component motion

**Aperture Problem Solution:**
- MT integrates local motion signals
- Computes global motion direction

**Motion Transparency:**
- Can represent multiple motion directions simultaneously

**Structure from Motion:**
- Perceiving 3D shape from motion cues

**Clinical:**
- MT damage causes akinetopsia (motion blindness)
- World appears as series of snapshots

### Layer Connectivity Summary for MT

| Connection | Origin Layer | Target | Function |
|------------|--------------|--------|----------|
| From V1 | V1 Layer IVB, III | MT Layer IV | Direct motion input |
| From V2 | V2 Layer III (thick) | MT Layer IV | Motion, depth |
| To MST | MT Layer III | MST Layer IV | Optic flow |
| To LIP, VIP | MT Layer III | Parietal Layer IV | Motion for action |
| To V1, V2 | MT Layer V/VI | V1, V2 Layer I | Feedback |
| To FEF | MT Layer III | FEF | Eye movements |
| To pontine | Layer V | Pons → Cerebellum | Pursuit |

## Part 8: Visual Area MST (Medial Superior Temporal)

**Location:** Anterior to MT in the superior temporal sulcus; extends into parietal territory

**Function:** Higher-order motion processing. Computes optic flow, self-motion, heading direction. Critical for navigation and distinguishing self-motion from object motion.

### Within Occipital/Temporal Border

**From MT:**
- MT Layer III → MST Layer IV
- Direction and speed information

**To MT (Feedback):**
- Layer V/VI → MT Layer I

### With Parietal Lobe

**To VIP (Ventral Intraparietal):**
- Optic flow for self-motion

**To Area 7a:**
- Visual-spatial integration

**To LIP:**
- Motion for saccade planning

### With Frontal Lobe

**To FEF:**
- Motion for eye movement control

**To Premotor Cortex:**
- Motion for reaching/action

### With Vestibular System

**From/To PIVC (Parietal Insular Vestibular Cortex):**
- Integration of visual and vestibular motion signals
- Distinguishing self-motion from world motion

### Functional Properties

**Optic Flow:**
- Expansion/contraction patterns
- Rotation patterns
- Heading direction from flow

**Pursuit Eye Movements:**
- Signals for smooth pursuit
- Compensates for eye movements

**Self-Motion vs Object Motion:**
- Distinguishes own movement from external movement

**Large Receptive Fields:**
- Can cover entire visual field
- Global motion integration

## Part 9: Lateral Occipital Complex (LOC)

**Location:** Lateral surface of occipital lobe, extending into posterior temporal; anterior to V3

**Components:**
- **LO (Lateral Occipital):** More posterior
- **pFs (Posterior Fusiform):** More anterior, overlaps with fusiform

**Function:** Object recognition. Responds to shapes and objects more than textures or scrambled images. Part of ventral stream.

### Within Occipital/Temporal Border

**From V4:**
- V4 Layer III → LOC Layer IV
- Intermediate form features

**From V3 (ventral):**
- Form information

**To V4, V3 (Feedback):**
- Layer V/VI → Lower areas Layer I

### With Temporal Lobe

**To/From Inferior Temporal (IT):**
- Bidirectional
- Object processing pipeline

**To Fusiform Face Area (FFA):**
- When processing faces

**To Parahippocampal Place Area (PPA):**
- When processing scenes

### With Frontal Lobe

**From Prefrontal Cortex:**
- Top-down object expectations
- Category information

### Functional Properties

**Object Selectivity:**
- Responds to objects > textures
- Responds to shapes > scrambled

**Shape Processing:**
- Curvature, contour
- Size invariance beginning

**Object Parts:**
- Represents object components

**Category Processing:**
- Beginning of category-level representation

## Part 11: Functional Streams Summary

### Dorsal Stream ("Where/How")

**Path:** V1 → V2 (thick stripes) → V3 → V3A → MT → MST → Posterior Parietal

**Functions:**
- Motion perception
- Spatial location
- Depth perception
- Action guidance (reaching, grasping)
- Eye movement control

**Characteristics:**
- Fast processing
- Magnocellular dominated
- Large receptive fields
- Transient responses
- Less affected by attention (relatively)

### Ventral Stream ("What")

**Path:** V1 → V2 (thin + pale stripes) → V4 → IT (TEO, TE)

**Functions:**
- Object recognition
- Color perception
- Form perception
- Face recognition
- Scene recognition

**Characteristics:**
- Slower, more detailed processing
- Parvocellular dominated
- Smaller receptive fields
- Sustained responses
- Strongly affected by attention

### Stream Interaction

The streams are NOT completely separate:
- V3 contributes to both
- MT ↔ V4 connections exist
- Parietal ↔ Temporal connections
- Both streams send to prefrontal cortex
